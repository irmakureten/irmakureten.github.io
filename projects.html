Project #1: Arithmetic Calculator

Prompt

Write a program that asks the user to enter two numbers. The program should calculate and display:

  • The sum
  • The difference
  • The product
  • The quotient (if the second number is not zero; otherwise, show a warning)

Example Output:
Enter the first number: 12
Enter the second number: 5
First number: 12.0
Second number: 5.0
Sum: 17.0
Difference: 7.0
Product: 60.0
Quotient: 2.4

Java Code


import java.util.Scanner;

public class MyProgram
{
    public static void main(String[] args)
    {
        Scanner scanner = new Scanner(System.in);

        System.out.println("Enter the first number: ");
        double a = scanner.nextDouble();

        System.out.println("Enter the second number: ");
        double b = scanner.nextDouble();

        double s = a + b;
        double d = a - b;
        double p = a * b;

        System.out.println("First number: " + a);
        System.out.println("Second number: " + b);
        System.out.println("Sum: " + s);
        System.out.println("Difference: " + d);
        System.out.println("Product: " + p);

        if (b == 0)
            System.out.println("The second number should be something different than zero.");
        else {
            double q = a / b;
            System.out.println("Quotient: " + q);
        }
    }
}
