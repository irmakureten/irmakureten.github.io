Problem 1: Basic Math Operations

Write a program that asks the user to enter two numbers.
The program should calculate and display:
– The sum
– The difference
– The product
– The quotient (if the second number is not zero; otherwise, show a warning)

Sample Output:
Enter the first number: 12
Enter the second number: 5
Sum: 17.0 | Difference: 7.0 | Product: 60.0 | Quotient: 2.4 


import java.util.Scanner;
public class MyProgram {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        System.out.println("Enter the first number: ");
        double a = scanner.nextDouble();
        System.out.println("Enter the second number: ");
        double b = scanner.nextDouble();

        double s = a + b;
        double d = a - b;
        double p = a * b;
        double q = a / b;

        System.out.println("First number: " + a);
        System.out.println("Second number: " + b);
        System.out.println("Sum: " + s);
        System.out.println("Difference: " + d);
        System.out.println("Product: " + p);
        
        if (b == 0)
            System.out.println("The second number should be something different than zero.");
        else
            System.out.println("Quotient: " + q);
    }
}

Problem 3: Rocket to the Stars

A rocket is launched vertically from Earth.
The user enters the rocket’s mass (kg), engine force (N), and burn time (s).
The program calculates:
– Net acceleration = F / m
– Final velocity = a * t
– Altitude = 0.5 * a * t * t
It displays both decimal and integer (cast) values.
Bonus: ASCII rocket! 


import java.util.Scanner;
public class MyProgram {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        
        int mass = scanner.nextInt();
        int force = scanner.nextInt();
        int burn = scanner.nextInt();

        int a = force / mass;
        double acc = (double) force / mass;
        double vel = acc * burn;
        double alt = 0.5 * acc * burn * burn;

        System.out.println("Acceleration: " + acc);
        System.out.println("Velocity: " + vel);
        System.out.println("Altitude: " + alt);

        System.out.println("  /\\  ");
        System.out.println(" /  \\ ");
        System.out.println(" |==| ");
        System.out.println(" |  | ");
        System.out.println("/____\\");
        System.out.println("  **  ");
    }
}

Bonus Challenge: Newton’s Square Root

Without using Math.sqrt, compute the square root of a number N using Newton’s method:
- Start with x₀ = N / 2
- Compute x₁, x₂, and x₃ using:
x(k+1) = 1/2 * (x(k) + N / x(k))
- No loops or conditionals
Show decimal and integer result. 


import java.util.Scanner;
public class MyProgram {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        double N = scanner.nextDouble();

        double x0 = N / 2.0;
        double x1 = 0.5 * (x0 + N / x0);
        double x2 = 0.5 * (x1 + N / x1);
        double x3 = 0.5 * (x2 + N / x2);

        System.out.println("Approximate sqrt(N): " + x3);
        System.out.println("As integer: " + (int)x3);
    }
}

Princeton 29: Day of the Week Calculator

Write a program that takes a date (month, day, year) and prints the day of the week (0 = Sun, ..., 6 = Sat).
Use:
y0 = y − (14 − m) / 12
x = y0 + y0 / 4 − y0 / 100 + y0 / 400
m0 = m + 12 * ((14 − m) / 12) − 2
d0 = (d + x + 31 * m0 / 12) % 7 


import java.util.Scanner;
public class MyProgram {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        int m = scanner.nextInt();
        int d = scanner.nextInt();
        int y = scanner.nextInt();

        double y0 = y - (14 - m) / 12;
        double x = y0 + y0 / 4 - y0 / 100 + y0 / 400;
        double m0 = m + 12 * ((14 - m) / 12) - 2;
        double d0 = (d + x + 31 * m0 / 12) % 7;

        System.out.println((int) d0);
    }
}
