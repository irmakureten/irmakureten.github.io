Write a Java program that asks the user to enter two numbers. The program should calculate and display:

  • The sum
  • The difference

// Pseudocode:
// 1. Import Scanner to get user input
// 2. Ask for two numbers
// 3. Calculate and print sum, difference, product, and quotient
// 4. Display a warning if division by zero is attempted

import java.util.Scanner;

public class MyProgram {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        System.out.print("Enter the first number: ");
        double a = scanner.nextDouble();

        System.out.print("Enter the second number: ");
        double b = scanner.nextDouble();

        double s = a + b;
        double d = a - b;
        double p = a * b;
        double q = a / b;

        System.out.println("First number: " + a);
        System.out.println("Second number: " + b);
        System.out.println("Sum: " + s);
        System.out.println("Difference: " + d);
        System.out.println("Product: " + p);

        if (b == 0) {
            System.out.println("The second number should be something different than zero.");
        } else {
            System.out.println("Quotient: " + q);
        }
    }
}
